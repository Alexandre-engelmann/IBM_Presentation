Kubernetes labs

  1. Lab-1: Push an image of an app to IBM Cloud Container Registry and deploy an app to a cluster(old command bx now ibmcloud)
    • Push and image
      1. Login to IBM cloud CLI: ibmcloud login --sso
      2. Log in with your IBM Cloud credentials to be allow to push images to IBM Cloud Container Register: ibmcloud cr login
      3. Make a namespace in IBM Cloud Container Registry: ibmcloud cr namespace-add <namespace>
      4. Build a docker image: docker build --tag us.icr.io/<my_namespace>/hello-world .
